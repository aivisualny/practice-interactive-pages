W.A.LT (Window Attention Latent Transformer)

Transformer 확산 모델 2023

🔧 핵심 구조 및 기술

  • Transformer 기반 잠재 공간 비디오 확산 모델 - 영상과 이미지를 모두 인코딩하는 공통 잠재 공간
  • 윈도우 제한 자기어텐션 - 메모리 효율적인 고해상도 영상 생성
  • U-Net 대신 교차적 공간-시간 윈도우 어텐션 Transformer 사용
  • 통합 VQ-인코더 - 이미지/비디오 차원 축소로 효율성 증대
  • 모델 규모: W.A.LT-L (3억 1천만 파라미터), W.A.LT-XL (4억 6천만 파라미터)

📊 생성 품질 및 평가

  • UCF-101: W.A.LT-XL FVD 36±2 (이전 최고 MAGVIT-v2 58±2 대비 크게 개선)
  • Kinetics-600 프레임예측: W.A.LT-L FVD 3.3 (최저 기록)
  • ImageNet 256×256: FID 2.56, IS 215.1 (VDM++과 동급, 파라미터는 1/4)

✅ 장점

  • Transformer의 병렬성과 표현력을 영상 생성에 도입
  • 클래스 조건부 및 무조건 생성 모두에서 고품질 결과
  • 128×128부터 3단계 점진 업샘플링으로 고해상도 영상 생성
  • Classifier-free guidance 없이도 안정적인 품질
  • 이미지와 비디오 공동 학습으로 정적 묘사와 동역학 학습 모두 활용

❌ 단점

  • Conv U-Net 대비 높은 메모리 요구량과 훈련 비용
  • 윈도우 크기로 인한 국부적 상호작용 한계
  • GAN 대비 느린 추론 속도 (50 step 샘플링 필요)

🎯 조건 및 활용 환경

  • W.A.LT-XL: 클래스 레이블 조건에서 최고 성능
  • W.A.LT-L: 무조건 생성, 프레임 조건부 예측 등 다양한 설정
  • 텍스트 조건: 별도 파이프라인으로 128×128 생성 + 영상 초해상도
  • 클래스, 텍스트, 이전 프레임 등 모든 조건에서 강건한 성능

MAGVIT (Masked Generative Video Transformer)

Transformer VQ-VAE 2023

🔧 핵심 구조 및 기술

  • 3D 비디오 토크나이저 - 영상 클립을 3D 벡터 양자화로 시공간 토큰 시퀀스로 압축
  • 마스킹된 토큰 예측 Transformer - 양방향 Transformer가 다양한 패턴의 [MASK] 토큰을 채우도록 학습
  • 다중태스크 학습 - 10가지 이상의 영상 생성 태스크를 랜덤 혼합 학습
  • 비자기회귀적 토큰 생성 - 모든 토큰을 병렬로 예측하여 속도 혁신

📊 생성 품질 및 평가

  • Kinetics-600: FVD 9.9 (2023년 CVPR 등장 당시 최고 성능)
  • UCF-101: FVD ~76 (동시대 모델 대비 우수)
  • 속도 혁신: 확산 모델 대비 100배, Autoregressive 모델 대비 60배 빠름

✅ 장점

  • 범용성: 단일 모델로 영상 생성, 연장, 보간, 부분변환 등 10가지 과제
  • 효율성: 매우 빠른 추론 속도로 실시간에 가까운 생성
  • 병렬 처리: 토큰 시퀀스 변환으로 연산량 대폭 감소
  • 실용성: 거대 확산모델과 견줄 품질, 월등한 추론 속도

❌ 단점

  • VQ-토큰화로 인한 세부 화질 제한 (확산 모델 대비 덜 섬세)
  • 토큰 양자화로 인한 디테일 손실 및 부자연스러운 패턴 가능성
  • 1세대는 텍스트 조건 직접 처리 불가
  • 훈련 복잡도가 높고 대규모 데이터 학습 시 자원 소모
  • 코드북 크기, 토큰 길이 등 하이퍼파라미터 설정 복잡

🎯 조건 및 활용 환경

  • 무조건 생성과 클래스 조건 생성 모두 지원
  • 부분적 조건: 첫 몇 프레임 제공해 계속 생성, 영상 일부분 채우기
  • 영상 이어붙이기, 배경 고정 + 움직임 생성 등 응용
  • 해상도: 256² 정도로 시작, 별도 VQ 모델로 업샘플링 가능
  • 조건 마스크를 활용한 임의 조건 영상 생성 프레임워크

StyleSV (Style-based Video Synthesis)

GAN StyleGAN 기반 2023

🔧 핵심 구조 및 기술

  • StyleGAN3 기반 2단계 학습 - 이미지 생성 후 비디오로 파인튜닝
  • Alias-free 합성 - 연속 프레임 간 텍스처 고정 현상 해결
  • Temporal Shift Module (TSM) - 판별기에 시간적 변화 판별 능력 부여
  • B-스플라인 기반 모션 표현 - 무한 길이 영상 생성 가능
  • 저랭크 시간 변조 - 장시간 생성 시 주기적 반복 문제 완화

📊 생성 품질 및 평가

  • Sky Time-lapse: FVD 221.5 (StyleGAN-V 460.6 대비 절반 이하)
  • Taichi-HD: FVD16, FVD128 모두 대폭 개선
  • First-person Driving: 짧은/긴 구간 일관성 모두 향상
  • 무한 길이 생성: 수천 프레임 이어붙여도 자연스러운 연속성

✅ 장점

  • 높은 시각적 품질: Alias-free 생성기로 개별 프레임 해상도와 선명도 우수
  • 시간적 안정성: TSM 판별기로 물체 움직임 일관성 확보
  • 무한 길이 생성: 앞뒤 부분 질감/배경 연속성 유지
  • 빠른 추론: GAN 특성상 실시간 프레임 생성
  • 고해상도 지원: 256×256~512×512까지 확장 가능

❌ 단점

  • 비조건부 생성: 텍스트/라벨 조건 생성 미지원
  • 제한적 다양성: 특정 분야 데이터셋에 한정
  • 학습 불안정성: GAN 특유의 세심한 튜닝 필요
  • 모션 붕괴 위험: 경우에 따라 부자연스러운 움직임
  • 잔여 주기 패턴: 아주 긴 생성 시 완전히 해결되지 않음

🎯 조건 및 활용 환경

  • 무조건 영상 생성: 길이 조절 가능한 자연 풍경, 패턴 월페이퍼
  • 가상 타임랩스: 하늘, 구름, 자연 현상 등 무한 재생
  • VR/엔터테인먼트: 배경 영상 합성, 24fps 고품질 출력
  • 해상도: 256~512px 수준, 양질의 영상 생성
  • 제한사항: 텍스트 설명이나 카테고리 레이블 제어 불가

Make-A-Video vs CogVideo

Diffusion vs Transformer Text-to-Video 2022

🔧 핵심 구조 및 기술 비교

Make-A-Video (Meta AI)
  • 확산 모델 기반 - 사전 학습된 이미지 확산 모델 활용
  • 시공간 확산 구조 - Pseudo-3D Conv + Temporal Attention
  • 계단식 생성 파이프라인 - 64×64 → 256×768 → 프레임 보간
  • 텍스트-비디오 페어 불필요 - 이미지+동영상 데이터로 분리 학습
CogVideo (Tsinghua)
  • Transformer 기반 - 9억 파라미터 거대 모델
  • Autoregressive 생성 - 한글자씩 영상 그리기
  • 다단계 학습 - 멀티프레임레이트 전략
  • 사전학습 활용 - CogView2 고정 + Temporal Attention 추가

📊 생성 품질 및 평가

  • 인간 평가: Make-A-Video 80% 선호도 vs CogVideo 20%
  • MSR-VTT: Make-A-Video FVD 크게 감소
  • UCF-101 전이학습: Make-A-Video 월등히 낮은 FVD
  • 언어 지원: Make-A-Video 영어 기반, CogVideo 중국어 최적화

✅ Make-A-Video 장점

  • 훈련 효율성: 텍스트-비디오 페어 데이터 불필요
  • 창의적 다양성: 방대한 이미지-캡션 데이터 활용
  • 고해상도 출력: 768×768까지 단계적 생성
  • 다양한 응용: 이미지 애니메이션, 초해상도 등

❌ 공통 단점

  • 짧은 영상: 몇 초 분량만 생성 가능
  • 내러티브 한계: 장면 전환, 긴 상호작용 어려움
  • 해상도 트레이드오프: 복잡한 장면에서 품질 저하
  • 후처리 필요: 안정화, 보정 등 추가 작업

🎯 조건 및 활용 환경

  • Make-A-Video: 영어 자연어 설명으로 다양한 스타일 영상 생성
  • CogVideo: 중국어 기반, 번역기 거쳐 영어 입력도 가능
  • 적용 분야: 예술, 광고, 배경 영상 등 제한적 용도
  • 현재 상태: 연구 목적 공개, 오픈소스 제공
  • 품질 편차: 후처리 작업 필요, 사실적 인물/긴 대화 장면 어려움

TGAN-F (Temporal GAN-Fast)

GAN 경량 GAN 2019

🔧 핵심 구조 및 기술

  • LDVD(저차원 비디오 판별기)로 3D Conv 대신 2D+1D 분리합성곱 사용
  • 단일 GPU 환경에서도 안정적 학습
  • 프레임별+시계열 특징 결합, 구조 단순화

📊 생성 품질 및 평가

  • UCF-101(128x128, 16f) IS: 24~25 (TGAN: 11, MoCoGAN: 12)
  • FVD 기준 TGAN 대비 절반 이하
  • 경량 구성 대비 우수, 이후 대형 모델에 비해선 낮음

✅ 장점

  • 경량화, 안정화, 단일 GPU 학습 가능
  • 모드 붕괴 감소, GAN 훈련 효율성

❌ 단점

  • 고해상도/장기 영상 확장성 제한
  • 조건부 생성 미지원, GAN 한계 존재

🎯 조건 및 활용 환경

  • 무조건 생성, 단일 도메인 영상 합성
  • 실시간성, 교육/실험용 경량 GAN

TGANv2

GAN Progressive GAN 2020

🔧 핵심 구조 및 기술

  • Progressive Growing(점진적 성장) 기법 도입
  • 다중 generator 구조로 메모리 분산
  • 해상도/프레임 길이 점진적 증가

📊 생성 품질 및 평가

  • UCF-101(128x128, 16f) IS: 28.87 (DVD-GAN: 27.3, VideoGPT: 24.7)
  • FVD 수백 대 후반 (동시대 GAN 중 선도)
  • 복잡한 분포, 얼굴 영상 등에서 선명도 향상

✅ 장점

  • 스케일 확장성, 고해상도 합성 가능
  • 병렬 학습, 대용량 데이터 적용 용이

❌ 단점

  • 구조 복잡, 튜닝 난이도 높음
  • 조건부 생성 미지원, 최신 SOTA 대비 품질 한계

🎯 조건 및 활용 환경

  • 무조건 생성, 대규모 데이터셋 합성
  • 실시간 응용, 단일 도메인에 유리

Imagen (정적 프레임 baseline)

Diffusion 이미지 기반 2022

🔧 핵심 구조 및 기술

  • 초고화질 텍스트-투-이미지 확산 모델
  • 영상 생성 연구에서 baseline(정지영상)으로 사용
  • 단일 이미지 반복(6프레임 등)으로 영상처럼 취급

📊 생성 품질 및 평가

  • COCO FID 7.27, 인간 평가에서 실제 이미지와 동등
  • FVD 등 영상 평가지표에서는 매우 낮은 점수
  • 정지화면 화질은 최고, 모션은 전혀 없음

✅ 장점

  • 정지 프레임 화질 최고, 텍스트 일치도 우수
  • 추론 속도 빠름, 영상 데이터 없이도 사용 가능

❌ 단점

  • 모션/시간 일관성 전혀 없음
  • 실제 영상 생성에는 부적합, baseline 용도

🎯 조건 및 활용 환경

  • 정지화면이 허용되는 영상(슬라이드쇼 등)
  • 연구적 baseline, 영상 생성의 하한선 실험

PG-SWGAN-3D (Progressive Growing Sliced Wasserstein GAN-3D)

GAN Progressive GAN 2019

🔧 핵심 구조 및 기술

  • Progressive Growing GAN - 공간+시간 3D 네트워크 점진적 확대
  • Sliced Wasserstein GAN 손실 - 고차원 영상 분포 간 거리 효율적 계산
  • 저해상도-짧은 영상에서 고해상도-긴 영상으로 부드러운 학습
  • 무작위 판별 함수(슬라이스)들의 평균으로 Wasserstein 거리 근사

📊 생성 품질 및 평가

  • TrailerFaces 데이터셋: 사용자 선호도 90% 이상 (VGAN, MoCoGAN 대비)
  • FID 비교: PG-WGAN-3D 402 vs PG-SWGAN-3D 404 (근사치)
  • 사용자 평가: SWGAN 적용 시 54% vs 46% 선호도
  • 시간적 자연스러움: SWGAN이 더 높은 점수

✅ 장점

  • 안정적 고해상도 영상 합성: Progressive Growing + SW 손실 조합
  • 복잡한 표정 변화 포착: 눈 깜빡임, 입꼬리 움직임 등 자연스러움
  • 구현 간단: 별도 판별망 튜닝 없이 적용 가능
  • 학습 안정성: 모드 다양성 유지, 그래디언트 안정적 전달

❌ 단점

  • 아키텍처 혁신 부족: 손실함수 수준 개선에 머물러
  • Progressive Growing 복잡성: 해상도 단계별 잠재적 불안정성
  • 정량적 개선 미미: FID 개선폭이 크지 않음
  • 도메인 제한: 얼굴 등 제한된 도메인에서만 검증

🎯 조건 및 활용 환경

  • 비지도 영상 생성: 특정 도메인 데이터셋 학습 후 유사 영상 무한 생성
  • VR 아바타 얼굴 애니메이션: 셀카, 배우 클로즈업 영상 기반
  • 디지털 캐릭터 표정 합성: 256×256 해상도, 1~2초 분량
  • 실시간성: GAN 기반으로 빠른 추론, Progressive 구조로 다소 복잡

최신 동향 및 위치

확산 모델의 부상

  • W.A.LT, Make-A-Video 등 Transformer 기반 확산 모델이 SOTA 달성
  • 텍스트-비디오 생성에서 혁신적 성능 향상
  • 다양한 조건부 생성 능력 확보

GAN의 진화

  • StyleSV로 GAN 기반 영상 생성의 새로운 가능성 제시
  • 무한 길이 영상 생성과 시간적 안정성 개선
  • 특정 도메인에서 여전히 강력한 성능

속도 vs 품질 트레이드오프

  • MAGVIT의 초고속 추론 vs W.A.LT의 최고 품질
  • 실용적 응용을 위한 속도 최적화 모델 등장
  • 다목적 모델의 필요성 증가

PG-SWGAN-3D의 위치

  • 2019년 TrailerFaces 벤치마크 SOTA, 영상 GAN 기준 모델
  • 고해상도 영상도 GAN으로 가능하다는 증명
  • Wasserstein 대안손실의 효과 입증
  • 현재는 StyleGAN-V, StyleSV 등에 대체되었으나 특정 응용에서 참고 가치
