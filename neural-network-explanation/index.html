신경망은 어떻게 작동하나요?

신경망의 각 뉴런은 하나의 미니 의사결정자처럼 작동합니다. 고양이 사진을 인식하는 과정을 예로 들어보겠습니다:

  1. 입력층: 사진의 픽셀 정보를 받아들입니다.
  2. 은닉층: 각 뉴런이 특정 패턴(귀, 수염, 얼굴 모양 등)을 찾아냅니다.
  3. 출력층: 발견된 패턴들을 종합하여 "이것이 고양이인가?"에 대한 최종 확률을 계산합니다.
